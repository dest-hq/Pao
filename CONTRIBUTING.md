# Contributing Guide

1. Please use LLMs as little as possible to generate code
2. There are 2 things you can work on
   - **Bug fixes** see [issues](https://github.com/dest-hq/pao/issues)
   - **New features** see [ROADMAP](https://github.com/dest-hq/Pao/#Roadmap)
3. **Fork the repo** and create a branch for your changes
4. The code must compile `cargo build`
5. Format the code `cargo fmt`
6. Code must not be malicious
7. **Open a Pull Request** with a clear description

Questions? Open an issue or discussion

Thank you for contributing
